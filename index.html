Bem-vindo ao ProSimulador

Este simulador irá permitir que o usuário faça cálculos para no fim
obter o valor total do seu orçamento.

Instruções

Para começar a usar o ProSimulador, siga as instruções abaixo:

  1. Adicione os itens que deseja incluir no orçamento.
  2. Insira a quantidade e o preço de cada item.
  3. Clique em "Calcular" para obter o valor total do orçamento.
